# SafeRoutes

![SafeRoutes](public/saferoutes.png)

## Challenge Innovation Sécurité Routière 2026

This project is submitted for the **Challenge Innovation Sécurité Routière 2026** (Road Safety Innovation Challenge 2026).

Aimed at students, this competition rewards projects that improve road safety. A prize pool ranging from 3,000 to 12,000 euros is awarded to the three winning teams.

## Overview

SafeRoutes is a mobile application designed to anticipate risky road zones before incidents occur and provide discrete, useful alerts. The application serves as a safety layer for risk prevention and post-trip insights, not as a GPS navigation replacement.

## Tech Stack

- **Framework**: Next.js 14
- **Language**: TypeScript
- **Styling**: Tailwind CSS
- **Fonts**: Bricolage Grotesque (Google Fonts)

## Getting Started

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.
